# Cybersecurity Attack Detection using Machine Learning

This project implements a machine learning pipeline for cybersecurity attack detection using both static and streaming evaluation methods.

## Project Overview

The project includes:

- data exploration and preprocessing
- feature selection
- model training and evaluation
- static model analysis
- streaming model evaluation using window-based processing

## Models Used

- Gradient Boosting
- Gaussian Naive Bayes

The Gradient Boosting model achieved the best performance with accuracy around 82%.

## Files

- `static_model_training.ipynb` — static data preprocessing, feature selection, model training, and evaluation
- `streaming_model_evaluation.ipynb` — dynamic / streaming evaluation of the trained model
- `kafka_dataset.csv` — dataset used in the project
- `project_report.pdf` — written project report and methodology

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Matplotlib
- Docker
- Kafka
